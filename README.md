> **Tier 2 — Consumer Electronics Vertical** — This factory manufactures household kitchen appliances
> (refrigerators, microwaves, electric cookers, blenders and domestic appliances) for the Nigerian market,
> with expansion into West Africa and export to the African diaspora. See [MASTER_REPO_REF.md](./MASTER_REPO_REF.md)
> for traceability back to the [Coo-Kah-Doks](https://github.com/oumar-code/Coo-Kah-Doks) master repository.
